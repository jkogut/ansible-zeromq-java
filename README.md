0MQ with jzmq and Java examples playbook
===

### 1. Install zeromq
### 2. Download java zeromq bindings 
### 3. Configure, compile and install java zeromq
### 4. Create ld.conf.zeromq 
### 5. Update ldconfig
### 6. Download zguide examples
### 7. Compile and run Java 0MQ examples
